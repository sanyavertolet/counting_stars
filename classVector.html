Detailed Description

template<typename TValue>
class Vector< TValue >

Header containing Vector class declaration.

Author
sanyavertolet Template class that implements vector.

If TValue is bool, data is stored as uint64_t value, otherwise it is stored as std::map<StringIng, TValue>. If capacity is negative, vector is considered to be endless, otherwise it has maximum of capacity elements.

Template Parameters
TValuetype of stored data.
